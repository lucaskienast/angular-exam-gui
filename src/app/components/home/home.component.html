Welcome to examitron.io


This Angular examination webapp with a Java (Spring Boot) and H2 backend allows you to:

  1. create online exams
  2. take part in exams
  3. review exam results

Click on "New Exam" at the top right of the page to create a new exam.

Open the sidebar to view the list of all exams and to participate in or review them.

Info: Make sure the backend is running on port 8080. The backend project repo is available at: GitHub
